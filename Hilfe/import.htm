Import eines Chats

Ein Chat kann auf zwei Weisen eingelesen werden:

  1. direkt aus einer E-Mail heraus (aktuell nur für Microsoft Outlook)
  2. aus einer Datei in einem Verzeichnis

Wann immer möglich, solltet Ihr einen Chat über die E-Mail-Funktion einlesen und die zweite Alternative nur nutzen, wenn es gar nicht anders geht!


Standardverfahren

Zum Einlesen eines Chats per E-Mail zieht ihr die E-Mail mit dem Chat aus Outlook in einen beliebigen Ordner auf eurem Rechner. Anschließend wählt ihr im Auswerteprogramm den Punkt Lade Chat aus Outlook-Mail und öffnet die gerade gespeicherte Datei.


Alternative

Zum Einlesen eines Chats, der auf alternativem Weg auf den Rechner geladen wurde, wählt ihr im Auswerteprogramm den Punkt Lade Chat aus Datei und geht dann in das Verzeichnis, in dem die Daten gespeichert wurden:
